Concurrent Async Queue

Programming utilities

Concurrent Async Queue è una estensions della classe ConcurrentQueue del framework .NET. FOrnisce una interfaccia asincrona alla coda e dispone di metodi di accesso estesi che consentono di sospendere la thread chiamante fino a quando una entità specifica non è disponibile all'interno della coda.

Cryptoberry

Programming utilities

Cryptoberry è un gateway di sicurezza che consente di cifrare le immagini registrate talle telecamere di sorveglianza domestica prima di inviarle al cloud per l'archiviazione. Dispone anche in una app di visualizzazione che gestisce a decifratra delle immagini in modo del tutto trasparente all'utente.

Programming utilities

MVVM Light Async Messages è una estensione del meccanismo di scambio messaggi disponibile all'interno del framework MMVM Light che espone una interfaccia async per l'invio e la ricezione dei messaggi.
